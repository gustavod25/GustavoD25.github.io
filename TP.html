Intalacion de Linux

Para instalar Linux junto a Windows se requiere la imagen de la distribución que uno desea instalar booteada en un dispositivo de almacenamiento y la pc con el Windows instalado y además se debe de realizarla dos particiones adicional para la instalación del SO que se configuraran posteriormente.

Se debe de conectar el dispositivo de almacenamientos, ya sea USB o cd a la pc y al encenderla se debe de pulsar las teclas correspondiente a cada maquina para que inicie el booteo y seleccione donde se encuentre el instalador.

Al iniciarse se dará opciones y se selecciona instalar. Luego da paso a la configuración de idioma, distribución de teclado, zona horaria, usuarios, contraseñas y posteriormente muestra el preparador de discos donde se formatea las particiones a usar, de las cuales la primaria debe tener espacio suficiente para albergar la raíz del sistema y si se desea, mayor espacio para almacenamiento y debe de ser de tipo ext4. La Lógica para el área de intercambio(SWAP) que funciona como memoria RAM cuando la física se agote.

A continuación se procede con la instalación del sistema en dicha partición. Una vez finalizado mostrara que el sistema se instalo correctamente y pedirá que reinicie para su uso.
